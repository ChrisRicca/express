Redirect

The Redirect plugin provides Request#redirect() as well as automatically assigning Request#home and Request#back .

Basic Redirecting

The example below shows a simple redirect to http://google.com 

get('/search', function(){
  this.redirect('http://google.com')
})

Redirecting Back

The HTTP Referrer header is used to redirect a user back to the page they just came from. 

get('/', function(){
  this.redirect('back')
  // OR this.redirect(this.back)
})

Redirecting Home

Redirecting to the home page uses the same approach that redirecting "back" does. 

post('/login', function(){
  if (authenticationWasSuccessful)
    this.redirect('home')
})
